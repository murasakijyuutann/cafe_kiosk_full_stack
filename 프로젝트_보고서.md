# 카페 키오스크 프로젝트 계획서 요약본 - 4주

**프로젝트 기간:** 4주 (2025년 11월 11일 - 11월 27일)
**팀 규모:** 5명
**대상:** 초급 부트캠프 프로젝트
**제출 날짜:** 2025년 11월 27일

---

## 1. 프로젝트 개요

### 프로젝트 목표
고객이 다음을 수행할 수 있는 셀프 서비스 카페 키오스크 웹 애플리케이션:
- 카테고리별 메뉴 탐색 (커피, 디저트, 음료)
- 장바구니에 항목 추가
- 주문 및 주문 확인서 수령
- 주문 번호로 주문 조회

### 기술 스택
**백엔드:** Java 21, Spring Boot 3.5.6, MySQL 8.0, Spring Data JPA
**프론트엔드:** React 18, Vite, Bootstrap 5, Axios
**도구:** Git/GitHub, IntelliJ IDEA, VS Code

### 학습 목표
- 완전한 풀스택 애플리케이션 구축
- Git/GitHub로 팀 협업 실습
- RESTful API 아키텍처 구현
- 관계형 데이터베이스 설계 작업

---

## 2. 팀 구성 및 역할

| 팀원 | 역할 | 주요 작업 | 브랜치 |
|------------|------|------------|--------|
| **이재혁** | 데이터베이스 계층 | 엔티티 클래스, JPA 리포지토리 | `feature/database` |
| **강태성** | 비즈니스 로직 | DTO, 서비스 계층 | `feature/service` |
| **우선명** | REST API | 컨트롤러, 예외 처리 | `feature/controller` |
| **문수영** | 프론트엔드 (메뉴) | 메뉴 페이지, 카테고리 필터, UI | `feature/frontend-menu` |
| **김성진** | 프론트엔드 (장바구니) | 장바구니, 체크아웃, 주문 확인 | `feature/frontend-cart` |
